

## Pipy
 - Clean previous builds:
 ```console
rm -rf dist/ build/ *.egg-info
```

 - Build wheel and sdist:
```console
python -m build
```

 - Upload to PyPI:
```console
twine upload dist/*
```

## Docker
Build the docker
```console
docker build -t chipiron-x11 .
```

Run the docker with GUI
```console
sudo docker run --rm \
  -e DISPLAY=$DISPLAY \
  -v /tmp/.X11-unix:/tmp/.X11-unix \
  --network=host \
  -u $(id -u):$(id -g) \
  chipiron-x11
```

## Documentation

Clean previous builds
```console
rm -rf docs/_build
```

 Remove old .rst files auto-generated by sphinx-apidoc
```console
rm docs/chipiron*.rst
```


Re-generate .rst files from your Python package
```console
sphinx-apidoc -f -o docs chipiron
```

 Clean and rebuild docs
```console
cd docs
make clean
make html
```

Open the generated HTML in your browser
```console
xdg-open _build/html/index.html
```
